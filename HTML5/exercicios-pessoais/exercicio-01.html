Oque é html

HTML é uma linguagem de marcação para hiper texto usada para editar documentos que serão interpretados por navegadores web. É uma linguagem simples mas que precisa ser complementada por mais duas tecnologias que são javascript e css e assim você pode estrair todo o potencial frontende no desenvolvimento web.

Estrutura basica do html 5

Uma página html é estruturada por comando chamados de tags(etiquetas).
As tags são comandos os quais indicão ao navegador como ele deve interpretar determinado trecho de código

Estrutura das tags

As tags são indicads pelo simbolo de menor < nometag > e simbolo de maior.
Basicamente são dois os tipos:
Tags com fechamento que é exatamente o nome da tag antecipada pela barra (/) ficando da seguinte forma (< nometag> Informações < /nometag >);
Tags abertas São tags que tem um fechamento em si mesmas como exemplo a tag meta.No html5 as tags com fechamento em si mesmas podem ser escritas de duas maneiras que são opicionais < nome-da-tag />(com a barra) ou simplismente < nome-da-tag > (sem a barra) que é a mais atual sendo a primeira um formato usado em verções anteriores do html

Esqueleto de um arquivo html

O esqueleto de um arquivo html é sua estrutura mais basica e é composto pelas seguints tags no na ordem apresentada abaixo:

< !DOCTYPE html> * Indica para o navegador que o documento html é escrito na verção 5 do html
< html > *Inicia o documento html
< head > *Cabeça

Aque são colocados os comando referentes a cabeça do documento, geralmente as tags colocadas dentro desta tag não são visualizadas diretamente pelo usuario.
São comandos de configuração da pagina escritos em javascript e css.

< body> *Abertura do corpo

Desntro desta tag sao colocadas informações visualizadas diretamente pelo usuario.Textos, imagens, videos e gifes.

< /body> *Fechamento do corpo
< head > *Fechamento da cabeça
< /html> *Fechamento da tag html

Algumas tags da cabeça do html tag < head>

Como vimos a tag < head> é usada para adicionar comandos responsaveis pela interatividade da pagina tags tags relacionadas ao seu comportamento e codigos css e java script.

Algumas tags

1- < title>intitula a pagina aparece no canto superior esquerdo do navegador
2- < meta charset="utf-8"> da suporte a idiomas latinos compostos por acentos e caracteres especiais.
3- < link> e < scripts> codios de folhas de estilo css e java script.
