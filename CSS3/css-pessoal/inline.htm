Formas de aplicação do CSS3

Bem exixtem 3 formas de se aplicar estilos css em um documento html, a lista a baixo apresenta-os em ordem de prioridade de cascata que veremos com mais detales a frente.

  1. Inline style
  2. Internal style sheet
  3. External style sheet

Na sequencia detalharemos melhor acerca de cada uma delas.

Inline (na linha)

Como o propio nome sujere a forma de estilizar Inline significa dizer adicionar o código html diretamente na linha(tag no bory) a qual se deseja estilizear, seja uma cor ou fonte diferente.
Esta forma de aplicação do css deve ser usada de maneira moderada pois você não quer perder tempo por exemplo modificando tag por tag da sua pagina html.
