Bordas simples

Para usarmos bordas precisamos primeiramente definir três valores usando a propiedade border: espesura, tipo da borda e cor.
De uma forma mais simples você pode sintetizar tudo em um único comando onde temos a propiedade seguida dos valores na ordem apresentada acima mas sem as vírgulas.Veja:

border: 1px solid red; /*Este código define uma borda solida com 1 pixel de espesura na cor vermelha.*/

Essa é a forma simplificada de adicionar bordas. As bordas podem ser adicionadas em praticamente todas as tags HTML, se você abrir o códico fonte desta pagina no navegador verá a aplicação na pratica do uso das bordas.

Bordas completas

Bem, como visto no topico anterior temos uma forma simples de se criar bordas.Veremos agora um formato mais completo e que da ao desenvolvedor possibilidades mais avançadas.

Atravéz de propiedades mais especificas como border-color: border-width: e border-style: você pode personalizar de forma mais profunda a aparencia da borda. Vale lembrar que estas não são as unicas propiedades derivadas de bordas.Vamos exemplificar.

border-width: 1px 5px 10px 15px;
border-style: solid dotted dashed groove;
border-color: red blue gold green

Outra forma que podemos exemplificar é utilizando dois valores para cada propiedade onde teriamos uma sequencia diferente em que o primeiro valor correspondendo a parte superior e inferior da borda e o segundo as laterais. Veja.

Perceba tambem que existem varios estilos de bordas, logo a baixa confira a lista com todos os valores permitidos para o atributo border-style;

  • dotted - Define uma borda pontilhada
  • dashed - Define uma borda tracejada
  • solid - Define uma borda sólido
  • double - Define uma margem dupla
  • groove- Define uma borda sulcada 3D. O efeito depende do valor border-color
  • ridge- Define uma borda ridged 3D. O efeito depende do valor border-color
  • inset- Define uma borda inset 3D. O efeito depende do valor border-color
  • outset- Define uma borda início 3D. O efeito depende do valor border-color
  • none - Define sem fronteira
  • hidden - Define uma borda escondida
