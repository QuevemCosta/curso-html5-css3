Classe e IDs

Oque são IDs?

IDs são identificadores. Da mesma forma que o numero de uma poutrona serva para identifica-la como única dentro de um onibus os IDs são referências para pontos únicos no código.

Definição de um ID

# nome-do-elemento {}

O ID é usado somente uma vez por página e quando houver trabalhos com java script. A recomentação é que faça o uso de IDs para estruturar o sit.
Quandro precizar fzer formatações especificas para mais de um elemento em uma pagina aconcelha-se o uso das de uma classe.

Oque são classes?

Classe pode ser descrita como um conjunto de itens com caracteristicas em comum. Por exemplo homens pertencem a classe do sexo mascolino.

Aplicando classes

Definindo uma classe

Para definir uma classe usamos o caractere .(ponto), nome da class e {}
Exemplo:
.arbuto{ color:green; }

Usando uma classe

Escolha uma tag e adicione nela o atributo class="nome-classe" Perceba que não faz uso do ponto pois o ponto serve apenas para indicar a criação/definição de uma classe.
A ideia de formatar usando classes é uma forma mais especifica de estilisar, por outro lado o uso de seletores para uma formatação mais geral. Ou seja, usamos classe para exceções.
Voce pode ver essa diferença mostrando no navegador o código fonte desta pagina.
